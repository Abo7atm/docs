# Introduction

The following documentation provides you with the steps required to integrate **SuperTokens** in your application.
The information is structured in the context of implementing an **Email and Password** authentication flow.

## Quickstart

If you want to get started with **SuperTokens**, you can follow our [**main guide**](./quickstart). 
It will provide you with step by step instructions on how to setup a simple application.

### Example Apps

Additionally, if you want a faster way to spin up a working application, you can use our **example apps**.
Just run `npx create-supertokens-app@latest` and start the project on your local machine.
