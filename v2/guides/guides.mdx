# SuperTokens Documentation

#### Get an overview of SuperTokens's features and how to integrate them into your applications

**SuperTokens** is a flexible, open-source authentication solution that allows you to add a login flow to your application in minutes. 
It is designed to be customizable and extensible, so you can adjust it with ease in order to fit your specific use case. 
Throughout this documentation you will find detailed guides on how SuperTokens works, how to integrate it in your product and how to enable all the features that you need.

## Getting Started 

For a step by step guide on how to add SuperTokens to your existing application please follow our [Quickstart Section](/docs/guides#quickstart-selection).
In there you will find instructions on how to setup your project in order to sign up and authenticate users with our product.

### Example Apps

The fastest way to get an understanding of how SuperTokens works is to just install one of our example apps.
Just run `npx create-supertokens-app@latest` and start the project on your local machine. Use the [following page](/docs/cli-reference-guide) for extensive references on the app that you have installed.
