# List all roles

You can fetch all the roles that you have created

:::tip

You can also view all the roles from the user management dashboard. To know more about how to manage your user roles and permissions from user management dashboard see [this page](/docs/userdashboard/managing-user-roles-and-permissions)

:::

import UserRoles from "supertokens-node/recipe/userroles";

    // highlight-end
}
