# Introduction

## Overview

The **Passwordless** `recipe` provides a way of authenticating users through generated credentials like **Magic Links** or **One-Time Passwords**.
You can use it out of the box, with our **Pre-Built UI**, or implement your own interface through the available SDKs.

<img src="/img/passwordless-email-phone.png" style={{width: "70%"}} alt="Sign in form UI for passwordless login" />
