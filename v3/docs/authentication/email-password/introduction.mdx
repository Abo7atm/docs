# Introduction

## Overview

The **Email/Password** `recipe` provides a way of authenticating users with basic credentials.
You can use it out of the box, with our **Pre-Built UI**, or implement your own interface through the available SDKs.

<img src="/img/emailpassword-sign-in.png" style={{width: "70%"}} alt="Sign in form UI for email password login" />
