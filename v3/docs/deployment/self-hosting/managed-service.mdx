# Managed Service

## Creating a development environment âœ¨
- First, please [sign up](https://supertokens.com/auth)
- Select the auth method you want to use and follow the guided steps to integrate with our frontend and backend SDK if you have not done this already
<img alt="Integration with SuperTokens SDKs" width="500px" src="/img/saas/supertokens-core-quicksetup.png" />
- Select a region and click the deploy button:

:::tip
You should select a region that is closest to your backend.
:::

<img alt="Deploying SuperTokens Core" width="500px" src="/img/saas/supertokens-core-deploy.png" />

- After the deployment is complete the dashboard will look similar to this:
<img alt="Deployed SuperTokens Core" width="500px" src="/img/saas/supertokens-core-success.png" />

## Connecting the backend SDK with SuperTokens ðŸ”Œ
- Add the `connectionURI` and the `apiKey` shown on the dashboard to your code on the backend.

<BackendTabs.TabItem value="nodejs">

```tsx
import supertokens from "supertokens-node";

supertokens.init({
    // highlight-start
    supertokens: {
        connectionURI: "<CONNECTION_URI_HERE_FROM_THE_POPUP>",
        apiKey: "<API_KEY_HERE_FROM_THE_POPUP>"
    },
    // highlight-end
    appInfo: {
        apiDomain: "...",
        appName: "...",
        websiteDomain: "..."
    },
    recipeList: []
});
```

<BackendTabs.TabItem value="go">

```go
import "github.com/supertokens/supertokens-golang/supertokens"

func main() {
	supertokens.Init(supertokens.TypeInput{
        // highlight-start
		Supertokens: &supertokens.ConnectionInfo{
			ConnectionURI: "<CONNECTION_URI_HERE_FROM_THE_POPUP>",
			APIKey:        "<API_KEY_HERE_FROM_THE_POPUP>",
		},
        // highlight-end
	})
}

```

   framework='...', # type: ignore

      #...
   ]
